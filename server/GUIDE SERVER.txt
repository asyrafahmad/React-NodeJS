Guide

1. npm init --yes                                               = initiate package.json
2. npm install express                                          = for Rest API using NodeJS
3. npm install cors                                             = to connect front end (react) with backend (typescript)
4. npm instal mysql2/mongodb                                    = mysql2 (good for javascript intervention)
5. npm install nodemon                                          = for automtically run server if got changes  
6. npm install sequelize                                        = to install sequelize package (database connection)
7. npm install sequelize-cli                                    = to install sequelize cli
8. npx sequelize-cli init                                       = to initiate sequelize (config, migration, model)
9. npm install bcrypt                                           = for password encryption
10.npm install jsonwebtoken                                     = for user session token using JWToken




Note
1. JWT Token can be store in (Session Storage / Cookies (Secure) )
